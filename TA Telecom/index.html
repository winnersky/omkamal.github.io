Clustering Services Based on the following:

Step 1:

  • Pick coherets exceding 150 user at the first month.
  • Intergation till the 6th month for each coheret for each service.
  • We then compute the standard variation for those values to measure the variation.
  • We store also the number of coherets that we used for the above calculations.

Note: We ignore any services that has only one or two Coheret.

Step 2:

  • We use hierarchical clustering to divide those services into 7 clusters.

Clusters

  • x-axis: Standard Deviation for the intergated value, y-axis: Mean of the intergation value.
  • Size of the node: Number of Coherets, Color: The cluster id.
